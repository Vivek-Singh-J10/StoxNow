# StoxNow 📈

A full-stack stock trading dashboard application built using the MERN stack 
(MongoDB, Express.js, React, Node.js) — designed to simulate a real-world trading 
platform experience with portfolio tracking, watchlists, and order management.

⚠️ **Disclaimer:** This is a personal/educational project for demonstration purposes only. 
No real trading or financial transactions occur in this application.

## Features
- User authentication (signup/login)
- Portfolio/holdings dashboard
- Watchlist with mock live price updates
- Buy/Sell order UI (simulated, no real transactions)
- Basic charts for stock price trends

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB

## Why I built this
Built to practice full-stack development end-to-end — database schema design, 
REST API architecture, authentication, and a working real-time-style frontend.
